# Dasory Toolkit

안녕하세요! **Dasory Toolkit**에 오신 것을 환영합니다!

컴퓨터를 포맷하거나 새로운 PC를 샀을 때, "아... 필수 프로그램들을 또 언제 다 검색해서 다운로드하고 설치하지?" 하며 한숨 쉬어본 경험, 다들 한 번쯤 있으시죠? 
이 프로젝트는 그런 끔찍한 귀찮음을 한 방에 해결하기 위해 기획된 '나만의 원클릭 유틸리티 허브'입니다!

그냥 심플한 링크 모음집이냐고요? 아니요! 
개발하면서 이것저것 재미있고 실용적인 웹 기술들을 듬뿍 담아넣은 종합 장난감(?) 상자 같은 웹사이트입니다. 

---

## 주요 기능 (어떤 것들이 있나요?)

### 1. 필수 프로그램 원클릭 허브
Ninite, Everything, Geek Uninstaller, Malware Zero 등 PC 세팅과 쾌적한 관리에 절대 빠질 수 없는 '빛' 같은 프로그램들을 한 곳에 모았습니다. 클릭 한 번이면 공식 다운로드 페이지로 슝~ 이동합니다.

### 2. 초강력 무작위 비밀번호 생성기 (PassGen)
상단 메뉴바(GNB)의 'PassGen` 버튼을 누르면 부드러운 팝업 창이 나타납니다.

### 3. 24시간 실시간 방명록 (Daily Memo)
우측 하단의 메시지 둥근 버튼을 누르면 스르륵 열리는 소통 공간입니다.
실시간으로 연동되어서 누구나 익명으로 방명록을 남길 수 있습니다.
---

## 기술 스택 (어떻게 만들었나요?)

이 프로젝트는 무겁고 거창한 최신 프레임워크의 도움 없이, 순수한 웹의 기본기만으로 깎아 만들었습니다.
- **Frontend:** 순정 HTML5, CSS3, Vanilla JavaScript
- **Backend / DB:** Supabase
- **UI / Design:** FontAwesome 6 아이콘, 커스텀 Glassmorphism 스타일링

---

## 직접 만들고 싶다면?

이 프로젝트의 코드를 마음껏 가져가서 입맛에 맞게 고쳐 쓰셔도 좋습니다!
단, 껍데기뿐만 아니라 방명록이나 관리자 로그인 같은 서버 연동 기능을 직접 쓰시려면 다음 세팅이 필요합니다.

1. **Supabase 프로젝트 생성** 후 URL과 Public Anon Key를 발급받습니다.
2. `script.js` 상단에 본인의 Key 정보를 연결해 줍니다.
3. Supabase SQL Editor를 통해 `memos`와 `staff_content`라는 테이블을 만들고, 알맞은 RLS(Row Level Security) 접근 권한 규정을 걸어줍니다.
